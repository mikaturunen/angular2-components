FAQ

Is this a complete CSS library I could use for Angular 2?

I started working on this when Angular 2 went on beta for technical evaluation of Angular 2. It serves no real function other than showing one way of doing things during the beta and if you get some use of the components, please, do use them.

What components are supported and styled?

While I was working with Angular 2 I just thought of building a basic set of components I commonly see or personally use when building web applications. So you've got your buttons and such. Scroll down to see them.

How is the library tested?

Monkey testing (meaning manual testing that is) done mainly on Chrome as it's just one man effort to see what the technology does and not aiming to build a full library. The whole gallery just came about from that. There are some unit tests and specific tests for different components but the browser based support is not really validated to any degree.
